	flickr-api-keys.txt	
	README.md
	src	
	VARpedia.sh
	VARpedia_group25.jar

Open a bash terminal in the same directory as the script file named "VARpedia.sh", and type in the following commands:

#=======================================
# -> chmod 777 ./VARpedia.sh
# -> ./VARpedia.sh
